# работа с репозиторием

ресурсы сгруппированы по тематическим группам, каждую группу можно скомпилировать (`pack_here_res.bat` в конкретных папках) отдельно, либо пересобрать мод целиком (`pack_all_res.bat` в корне мода) 

вы можете удалять \_res папки в рабочем билде для ускорения сборки, но **не допускайте их удаления из общего репозитория**.

для отслеживания изменений крайне рекомендуется использовать текстовые форматы.

`git_story_maps.bat` - извлекает данные об объектах и скрипте внутри .mob файлов
